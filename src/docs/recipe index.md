# Creating Your First (Alphabetical) Index #
1. Under `Zip Recipes` menu click `Recipe Indexes`:
 <img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/8c275e9d9e63f716a0fd9963b185a1a747d6a79f_1_505x500.png" width="505" height="500">

2. Click `Create New Recipe Index`:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/46f6f920abafa76633afe1ecf42379af3bb924a2_1_634x500.png" width="634" height="500">

3. Click `Save`. This will create an `alphabetical` index:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/4b7cadab1c8443418ce5f0024488c10a3dc30939_1_488x500.png" width="488" height="500">

5. Now, let's show the index in a page. Copy the shortcode by highlighting, right clicking on it and then clicking `Copy`:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/ac68b1e8a5ae82d5d35783e76db4fe3522631855_1_690x393.png" width="690" height="393">

5. Create  a new Page by clicking `Pages > Add New` menu:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/a2a74bf932536ead1941b2d22c63e31ce303d791.png" width="690" height="364">

6. Add a title like `Recipe Index` and then paste the shortcode in the body:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/5fe4f6e69c6309b2a9f6df3df79bddd7cca2b114.png" width="690" height="345">

8. Publish it.
9. View the published page.
10. You should see your first index page. As you can see it's categorized alphabetically:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/346d8dd548cd44936aaca35feae8f93a1d0bd330_1_690x475.png" width="690" height="475">

# Creating a Category-based Index

1. Under `Zip Recipes` menu click `Recipe Indexes`:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/8c275e9d9e63f716a0fd9963b185a1a747d6a79f_1_505x500.png" width="505" height="500">
 
2. Click `Create New Recipe Index`:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/46f6f920abafa76633afe1ecf42379af3bb924a2.png" width="634" height="500">

3. Select `Recipe category` under `Group recipes using`
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/914dfa3b536a83edf38ed52147c8f326936a2ceb.png" width="690" height="477">

3. Click `Save`. This will create a `category` index.
4. Copy the shortcode by highlighting, right clicking on it and then clicking `Copy`:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/7e4acb00275e3b82f055f7cee0c1d906296261af_1_690x366.png" width="690" height="366"> 

5. Create  a new Page by clicking `Pages > Add New` menu:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/0dedac7e3b4a9837a8cc8939b6348f01344efba2.png" width="690" height="329">

6. Add a title like `Category Index` and paste the shortcode here:
<img src="http://forum.ziprecipes.net/uploads/default/optimized/1X/b3dc6efaf3918b321ad4c1492d737869de96629f_1_690x471.png" width="690" height="471">

8. Publish it.
9. View the published page.
10. Now you'll see you recipes grouped by category that they were saved in:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/ff8d95e2535e1303c62bf429246c60c12bfab42d.png" width="690" height="475">
For example, my `Chicken Paprika` recipe has the category `Hungarian` and it shows up under `Hungarian` in the Index page:
<img src="http://forum.ziprecipes.net/uploads/default/original/1X/3632c84b3d5c11c8453204d138f21773240fcdce.png" width="690" height="393">
